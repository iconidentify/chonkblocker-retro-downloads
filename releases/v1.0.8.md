# Chonk Blocker Retro 1.0.8

Replaces Uptown Circuit with a new night-drive funk arrangement and ships the refreshed cartridge in every desktop edition.

## What's in this release

- Replace Uptown Circuit with a new 32-bar night-drive funk arrangement in the cartridge soundtrack and hidden sound lab.
- Bundle the refreshed cartridge into the macOS, Windows, and Linux desktop editions, which had shipped the 1.0.6 cartridge in 1.0.7.
- Carry the 1.0.7 Linux desktop fixes unchanged: stable focus and pause behavior on Wayland compositors, resilient audio across focus changes, and correct fullscreen scaling.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
