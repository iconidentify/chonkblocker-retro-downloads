# Chonk Blocker Retro 1.0.2

A cartridge update that adds a hidden, native 16-bit sound lab for auditioning the complete Chonk Blocker soundtrack and effects catalog.

## What's in this release

- Open the hidden Chonk Lab from the SNES title menu by holding L + R + Select, then pressing Start.
- Browse and play all ten named Terrific Audio music tracks directly from the cartridge.
- Switch to an isolated sound-effects catalog containing all sixteen named effects, with scrolling, playback status, and a global silence control.
- Exit cleanly back to the normal title menu; the utility shell is structured to host additional internal cartridge tools in future updates.
- Ship the same approved 256 KiB ROM in the browser player, standalone download, and signed macOS and Windows editions, with a Linux AppImage and SHA-256 checksums.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
