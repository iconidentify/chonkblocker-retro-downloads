# Chonk Blocker Retro 1.0.3

A desktop performance release with exact-latch controls, explicit GPU presentation, full CRT effects, and resilient hardware-shaped audio.

## What's in this release

- Sample keyboard and controller state at the emulated SNES hardware latch, with a native input path that bypasses Swing's repaint queue.
- Present through an explicit OpenGL 3 backend with calibrated late submission, adaptive frame-delay timing, and no driver-queued video frame.
- Move the full CRT treatment to a multi-pass linear-light shader pipeline with scanlines, phosphor masks, convergence, composite bleed, persistence, bloom, curvature, glare, vignette, gamma, and dither.
- Rebuild S-DSP playback around four-point Gaussian interpolation and continuous resampling, then feed a clock-adaptive host audio queue without discontinuous sample dropping.
- Exercise raw and packaged GPU, focus-resume, frame-age, timing, and audio-recovery gates on macOS, Windows, and Linux before publication.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
