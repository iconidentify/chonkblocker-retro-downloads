# Chonk Blocker Retro 1.0.9

Shows the Chonk Blocker icon in the running window, taskbar, and dock instead of the bundled runtime's default logo.

## What's in this release

- Apply the product icon to the game window at runtime, so the Windows title bar, Alt-Tab switcher, and taskbar and Linux docks show Chonk Blocker instead of the JetBrains Runtime logo.
- Set the macOS Dock icon from the running application as well as the bundle.
- Ship the 1.0.8 cartridge and desktop behavior unchanged.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
