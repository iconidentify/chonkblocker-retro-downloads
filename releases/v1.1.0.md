# Chonk Blocker Retro 1.1.0

An elegant way out: closing the game now opens a cabinet-styled confirmation instead of quitting instantly, with a quit action reachable from settings and keyboard shortcuts.

## What's in this release

- Open a Leave the Arcade confirmation sheet when the window is closed, Alt+F4 or Ctrl+Q is pressed (⌘Q on macOS), or the new QUIT GAME button in settings is used — the game no longer ends without asking.
- Keep playing is the safe default: Enter chooses the highlighted action, arrow keys switch between Keep Playing and Exit Game, and Escape always returns to the game.
- Pause emulation and mute audio while the sheet is open, and return keyboard focus to the game seamlessly on resume.
- Add a quit path reachable in fullscreen, where the window's close button is not available.
- Ship the 1.0.8 cartridge unchanged.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
