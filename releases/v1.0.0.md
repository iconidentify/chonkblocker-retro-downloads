# Chonk Blocker Retro 1.0.0

The initial free release of Chonk Blocker rebuilt as a native 16-bit Super Nintendo game, plus focused desktop editions that open straight into the cartridge.

## What's in this release

- Play the standalone 256 KiB LoROM cartridge in compatible SNES hardware, flash cartridges, or your emulator of choice.
- Arcade and Endless modes recreate Chonk Blocker's gem physics, chains, garbage pressure, characters, nine-stage ladder, and score chase in native 65816 assembly.
- The Terrific Audio Driver powers ten original MML songs, responsive sound effects, BRR instruments, DSP echo, and danger-tempo shifts on the SNES audio hardware.
- The free macOS, Windows, and Linux editions bundle the same approved ROM with keyboard and controller support, fullscreen play, low-latency audio, and optional CRT presentation.
- macOS is Developer ID signed and notarized; Windows executables use Azure Trusted Signing; every download includes a SHA-256 checksum.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
