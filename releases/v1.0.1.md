# Chonk Blocker Retro 1.0.1

A desktop responsiveness update that makes keyboard input, CRT presentation, and audio feel substantially more immediate while preserving the approved SNES cartridge.

## What's in this release

- Present the newest emulated frame immediately so Start, movement, and block rotation no longer wait behind stale CRT frames.
- Rebuild CRT presentation around cached scanlines, phosphor masks, glass shading, bloom, and persistence that stay comfortably inside the SNES frame budget.
- Reduce and stabilize desktop audio buffering for faster audible feedback without allowing long-session device-clock drift to accumulate latency.
- Move the animated cabinet surround off the Swing input thread and coalesce repaint requests to keep keyboard handling responsive.
- Add opt-in frame-age, paint, input-to-paint, and audio telemetry plus regression coverage for the new low-latency pipeline.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
