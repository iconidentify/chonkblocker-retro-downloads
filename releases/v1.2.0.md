# Chonk Blocker Retro 1.2.0

Full couch control: navigate everything with a controller, with pad-family button glyphs, Steam Deck detection, and controller hot-plug fixed.

## What's in this release

- Navigate the whole app from the couch: SELECT+START opens settings, the d-pad moves focus, shoulders switch tabs, the bottom face button confirms, and the right face button backs out — on the settings sheet and the exit confirmation alike.
- Show button prompts as original vector glyphs that match the connected pad — Xbox, PlayStation, Switch, or Steam Deck — with keyboard keycaps when no controller is in use.
- Detect controllers plugged in after launch; hot-plug previously required a restart because SDL's device events were never serviced, which also fixes the RESCAN CONTROLLERS button.
- Launch fullscreen with pad-first prompts on Steam Deck and gamescope sessions, and honor a distribution-channel switch (direct / steam / mac-app-store) like the flagship game.
- Add Steam packaging scaffolding (Deck-safe AppImage launcher, default Steam Input profile, SteamPipe publisher) and the Mac App Store entitlement baseline — see docs/steam-and-mac-app-store.md for the remaining owner steps.
- Ship the 1.0.8 cartridge unchanged.

## Four ways to play

- **Standalone SNES ROM** — a native 16-bit cartridge image for compatible hardware and emulators.
- **macOS** — Apple-silicon DMG, Developer ID signed and notarized.
- **Windows** — x64 desktop application with Azure-signed executables.
- **Linux** — x86-64 AppImage.

Every asset has a sibling SHA-256 checksum. The desktop editions contain the exact standalone ROM published here.
